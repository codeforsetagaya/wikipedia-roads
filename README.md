# ウィキペディア街道プロジェクト

旧街道沿いの名所旧跡などの歴史・文化情報をインターネット上の百科事典「ウィキペディア」に掲載して世界に発信する、ウィキペディア街道プロジェクト用のレポジトリです。

---

## 街道
- 大山道（矢倉沢往還）
- 登戸道（津久井往還）
- 滝坂道（甲州中出道）
